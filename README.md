# Gunflint Lodge — Daily Operations Dashboard

A real-time operational dashboard for Gunflint Lodge & Outfitters displaying daily revenue, occupancy, and performance metrics.

## Features

- **Tonight's Overview** — Current occupancy %, rooms sold, ADR, and RevPAR
- **Revenue Breakdown** — Room, F&B, and gift shop revenue for the day
- **Guest Value** — Total spend per guest and person count
- **MTD vs Last Year** — Occupancy, room revenue, and ADR comparisons
- **F&B & Gift Shop Alerts** — Performance indicators vs. prior year
- **7-Day Forecast** — Occupancy projections for upcoming week
- **Data Status** — Source system health indicators

## Data Sources

- **RoomMaster** — Lodging occupancy and revenue
- **Aireus POS** — Restaurant and bar sales
- **Flybook** — Activity and outfitting data (pending integration)

## Auto-Updates

Dashboard refreshes automatically every 15 minutes from `operational-data-latest.json`.
